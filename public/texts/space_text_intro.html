Pour compléter ma base de données, j’ai créé une seconde collection : la collection « spaces » (espaces). Cette collection comporte une copie du corpus d’origine. Cependant, cette version diffère de l’original, en ceci qu’elle comporte une version enrichie de chaque documentaire. Pour enrichir les données originalement comprises dans un documentaire, j’ai créé une nouvelle entité : l’entité « Space » (space). Cette entité est composite, elle comporte en résumé, une part restreinte des informations d’origine et une référence vers le documentaire original. L’entité comporte ensuite trois listes : 1. la liste de l’ensemble des lieux répertoriés dans le reportage, tout type confondu. 2. La liste des pays répertoriés dans le reportage et 3. la liste des villes répertoriées dans le reportage. Toutes ces listes comportent des références à des lieux, lieux qui se trouvent désormais dans la collection « locations ». Ces listes permettent d’une part de faire le distinguo entre les différents types de lieux mais elles permettent surtout d’avoir accès à des données enrichies autour des lieux. Le documentaire contient désormais un nom de lieux mais aussi l’ensemble des informations qui ont un rapport avec ce lieu puisqu’il fait référence à un élément dans la collection « locations ».
